# 停车场管理系统简介 java568

## 项目概述

停车场管理系统是一款基于B/S架构和MVC模式的web应用程序，采用前后端分离的开发模式，旨在提供便捷、高效的停车场管理解决方案。

## 技术栈

- **开发语言：** Java
- **数据库：** Mysql
- **后端框架：** SpringBoot、Mybatis-Plus
- **前端框架：** Vue
- **开发工具：** IDEA、JDK1.8、Maven

## 系统角色

- **普通用户：** 可进行个人信息管理、预定停车位、缴费信息管理等操作。
- **管理员用户：** 可进行用户信息管理、车辆信息管理、停泊车辆管理、车辆进出管理等功能操作。

## 运行环境

- **操作系统：** Windows

## 功能模块

- **个人信息管理：** 管理用户的个人资料。
- **停车位预定：** 预定停车位，并进行管理。
- **缴费信息管理：** 管理用户的缴费记录。
- **用户信息管理：** 管理系统用户的相关信息。
- **车辆信息管理：** 管理车辆的基础信息。
- **停泊车辆管理：** 管理车辆的停泊状态。
- **车辆进出管理：** 记录和管理车辆的进出情况。
